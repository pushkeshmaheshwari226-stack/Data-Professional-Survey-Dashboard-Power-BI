# Data Professional Survey Dashboard | Power BI

An interactive Power BI dashboard built to analyze survey responses from data professionals worldwide, uncovering trends in salaries, job satisfaction, programming language preferences, and career transitions.

## 📌 Project Overview

This project uses a real-world survey dataset of 630+ data professionals to answer key questions about the data industry:

- What do different data roles earn on average?
- How hard is it to break into the data field?
- Which programming languages are most popular?
- How satisfied are data professionals with their salary and work-life balance?

## 📊 Dashboard Highlights

| Visual | Insight |
|--------|---------|
| **Bar Chart** | Average salary by job title |
| **Bar Chart** | Favorite programming language by role |
| **Treemap** | Survey takers by country |
| **Donut Chart** | Difficulty breaking into data |
| **Gauge (x2)** | Avg. happiness — Salary & Work/Life Balance |
| **KPI Cards** | Total respondents & Average age |

## 🛠️ Tools & Skills

- Power BI Desktop
- Power Query (data cleaning & transformation)
- DAX (calculated columns & measures)
- Data Visualization & Dashboard Design

## 📁 Dataset

Survey of 630+ data professionals covering roles, salaries, programming language preferences, job satisfaction scores, and demographics.  
Source: Alex The Analyst — Data Analyst Bootcamp (2026)
